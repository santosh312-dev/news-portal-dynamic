# 📰 News Portal Dynamic

## 📌 Overview

**News Portal Dynamic** is a responsive web application that fetches and displays real-time news using JavaScript API integration. The project uses modular JavaScript files (`app.js` and `script.js`) to manage API requests, search functionality, and dynamic UI rendering.

It demonstrates strong understanding of **API handling, modular JavaScript structure, and DOM manipulation**.

---

## 🚀 Features

* 🔍 **Search News** – Find articles by keywords instantly
* 🌐 **Live News API Integration** – Fetches real-time news data
* 📰 **Dynamic News Rendering** – Displays news without page reload
* ⚡ **Modular JavaScript Structure** – Organized logic using `app.js` and `script.js`
* 📱 **Responsive Design** – Works across all screen sizes
* 🧩 **Clean UI Layout** – Card-based structured news display

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Functionality
* **REST API** – Fetching live news data

```
📁 news-portal-dynamic
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 app.js        # Handles core logic / API configuration
 ┣ 📄 script.js     # Handles UI updates & event handling
 ┗ 📄 README.md
```

---

## ⚙️ How It Works

1. `app.js` manages API requests and handles fetching news data.
2. `script.js` processes the data and updates the UI dynamically.
3. News articles are displayed as cards with title, image, and description.
4. Search input triggers new API calls and updates results instantly.

1. **Clone the repository**
